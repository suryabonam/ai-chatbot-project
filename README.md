# AI Chat Bot

An intelligent chatbot application that allows users to ask questions and receive AI-generated responses in real time.

## Features

* Real-time chat interface
* AI-powered responses
* Responsive design
* User-friendly interface
* Backend API integration

## Technologies Used

### Frontend

* React.js
* JavaScript
* HTML
* CSS

### Backend

* Python
* Flask

## Installation

1. Clone the repository

2. Install frontend dependencies

```bash
npm install
```

3. Install backend dependencies

```bash
pip install -r requirements.txt
```

4. Start the backend server

```bash
python app.py
```

5. Start the frontend application

```bash
npm start
```
